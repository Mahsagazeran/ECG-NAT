# ECG-NAT

## ECG-NAT: A Self-supervised Neighborhood Attention Transformer for Multi-lead ECG Classification

------

## About

This repository will contain the code, trained models, and datasets for our paper "ECG-NAT: A Self-supervised Neighborhood Attention Transformer for Multi-lead ECG Classification" .

**The code, trained models, and data will be made publicly available upon paper acceptance.**
